## 🚀 Introduction
This guide covers the **fundamental concepts of Git**, including repositories, initialization, staging, committing, and viewing history.

---

## 📂 What is a Git Repository?
A **repository (repo)** is a project folder tracked by Git with its own history.

---

## ⚙️ Initialize a Repository
```bash
mkdir MyProject
cd MyProject
git init
```

---

## 🔍 Check Repository Status
```bash
git status
```

---

## 📦 Git Workflow
Working Directory → Staging Area → Repository

---

## ➕ Stage Changes
```bash
git add file.txt
```

Add all files:
```bash
git add .
```

---

## ✅ Commit Changes
```bash
git commit -m "Your message"
```

---

## 📜 View Commit History
```bash
git log
```

---

## 🔁 Workflow Loop
Work → Add → Commit → Repeat

---

## 🎯 Key Takeaways
- Git tracks history
- Commits = snapshots
- Use meaningful messages
- Practice regularly

---

✅ Happy Learning Git! 🚀
